# Roll a Ball 
The project was made using the unity tutorial provided. It worked as a very good introduction to Unity, giving a basic but good understanding of how to use the software.

Firstly, some setup was made. A plane was added together with a ball, together with a new material to make the ball red. The next natural step was to make the player move, and later on to make the camera follow the player. These initial steps were fairly easy to follow and implement.

Increasing in complexity, it was time to create an actually playable environment, by adding walls to the plane (to ensure the player could not jump to the void) and adding collectibles that the player would need to “grab” in order to finish the game. I personally found it very enjoyable to work with both the animation of the collectibles and the prefab creation and placing across the map.

Once that was done, a display for the score was added, as well as an end game message. I found this a bit tricky at the beginning since the concept of the Canvas and the placing of the text seems a bit confusing for me, but after watching the whole tutorial and experience with it, it was successfully implemented.

Lastly, some basic AI was introduced. This AI would be following the player constantly, using a NavMesh to guide it through the map. A final build for the game was made, ending the tutorial

The game was not further expanded, and it main changes from the tutorial were different colors for the entities, different animations for the collectibles and a different map layout.
